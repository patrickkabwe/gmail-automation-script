## GMAIL AUTOMATION SCRIPT

A simple node script that deletes gmail emails by label (spam,trash...)

## Prerequisites

- Google account
- Gmail API
- Node.js
- Yarn

##### NOTE: 
- make sure to enable the Gmail API in your google account and download the keys.json file

## Installation

```bash
pnpm install
```

## Usage

```bash
pnpm start
```
